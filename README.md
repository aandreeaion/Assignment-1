# Assignment-1
The dataset contains a cvs file containing the English grades and scores for 50 students from CIS, Linguistics, IS or other.
The data was retrieved from a previous Statistics course.
## data.csv
The following columns exist in the dataset:
- participant
- sex
- bl_edu
- study
- english_grade
- english_score


| Column Name  | Description | Data Type |
| ------------- | ------------- | ------------- |
| participant  | contains the number of the participant, anonymous  | number |
| sex  | indicates the sex of the participant  | text |
| bl_edu  | specifies the student has bilingual education  | text |
| study  | shows which study programme the student is following | text |
| english_grade  | shows the grade from an eglish test  | number |
| english_score  | shows  the average score for the course | number |
